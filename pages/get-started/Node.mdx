# Node.js

### What is Node.js?

_Node.js is an open-source and cross-platform JavaScript runtime environment. It is a popular tool for almost any kind of project! Node.js runs the V8 JavaScript engine, the core of Google Chrome, outside of the browser. This allows Node.js to be very performant._

<a href="https://nodejs.org/en/download/" target="_blank">
  Node.js Download
</a>

<p>After installing Node.js, open your terminal and enter in the following ecommands:</p>
  <ol>
    <li>
      <code>npm install</code> to install all the packages
    </li>
    <li>
      <code>npm run dev</code> to start the development server
    </li>
    <li>
      After all the packages are installed, and the development server is running, you can open your browser and navigate to <code>http://localhost:3000</code>
    </li>
  </ol>
