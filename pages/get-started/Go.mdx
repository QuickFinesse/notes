# Go

### What is Go?

_Go, also known as Golang, is an open-source, compiled, and statically typed programming language designed by Google. It is built to be simple, high-performing, readable, and efficient._
<hr>
</hr>
<h3>Steps to install Go:</h3>
<ol>
  <li>Download <a href="https://go.dev/doc/install">Go</a>  and run the installation file</li>
  <li>Once you've completed the installation, open a new terminal window and verify that you've installed Go by typing the following command:</li>
  <h3>Windows & Mac: <code> go version</code>  <br></br><br></br>
  Linux: <a href="https://go.dev/doc/install">Installation Guide</a></h3> <br></br>
  <li>Once you've verified that Go is installed, feel free to start learing Go using this <a href= "https://go.dev/doc/tutorial/getting-started"> tutorial </a> </li>
</ol>
